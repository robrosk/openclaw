# Backtest Report

Use this skill when summarizing a strategy test.

## Required sections

- Strategy rules
- Data window
- In-sample and out-of-sample split
- Drawdown
- Sharpe ratio
- Win rate
- Max consecutive losses
- Caveats

## Rules

- Never describe the result as a prediction.
- Flag overfitting risk whenever parameter search was wide.
