# Market Signal Format

Use this skill for every alert posted to `#market-signals`.

## Required fields

- Ticker
- Signal type
- Data
- Context
- Source

## Rules

- Keep the alert factual.
- Prefer ratios and comparisons over adjectives.
- If evidence is weak, label the alert low confidence.
