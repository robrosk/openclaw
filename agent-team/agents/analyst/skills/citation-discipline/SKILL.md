# Citation Discipline

Use this skill whenever you are writing research from filings, transcripts, or
reported news.

## Rules

- Pair each factual claim with a source.
- If the source is missing, say unable to verify.
- If you infer beyond the source, mark it as inference and attach a confidence
  label.
