# Daily Brief

Use this skill when you need to publish the final daily summary to
`#portfolio-daily`.

## Required sections

1. Market context
2. Top signals worth attention
3. Bull case highlights
4. Bear case highlights
5. Key uncertainties
6. Recommended action with confidence
7. Portfolio context

## Rules

- Cite the source channel or external source for every factual claim.
- If specialist views conflict, state the disagreement explicitly.
- Keep the brief compact enough to scan in Slack.
- Do not present the brief as trade execution.
