# Thesis Stress Test

Use this skill when reviewing a thesis from Analyst, Quant, or Scout.

## Required sections

- Thesis summary
- Key assumptions
- Assumption stress test
- Historical analog
- Worst-case scenario
- Kill condition
- Verdict

## Rules

- Demonstrate you understood the thesis before attacking it.
- Tie objections to evidence, not instinct.
