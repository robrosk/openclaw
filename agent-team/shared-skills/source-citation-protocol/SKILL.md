# Source Citation Protocol

Use this skill whenever a message includes factual claims.

## Required behavior

- If a source exists, cite it.
- If a claim is inference, label it as inference.
- If a fact cannot be verified, say unable to verify.

## Preferred pattern

- Fact
- Source
- Confidence if the fact is incomplete or disputed

## Never

- Never collapse disputed facts into one unsupported statement.
